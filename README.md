GovOn helps you handle public request tasks with AI on your own Windows PC. It works without an internet connection and uses a smaller AI model tuned for real-world use. This guide will help you download the software and start using it.

## 📋 What is GovOn?

GovOn is a tool designed to process and analyze public service requests quickly and privately. It uses a light version of a large language model (LLM) to work on your device. The system reduces the need for internet access while helping with tasks like reading and understanding requests.

The project focuses on practical use in industry settings. It is fine-tuned to perform well in real work environments. It helps build skills for staff who handle citizen inquiries.

Key ideas behind GovOn include:

- On-device AI without needing the cloud  
- Lightweight and efficient AI models  
- Fine-tuned language models for better accuracy  
- Tools for managing public service requests  

## 💻 System Requirements

Before you begin, make sure your PC meets these requirements:

- Internet connection (only needed for the first download)

## 🔧 Using GovOn

The main screen lets you load public request files and analyze them with AI.

### Load a Request File

- Click "Open File"  
- Select a text file containing your public request  
- The file content will appear in the app

### Start Analysis

- Click the "Analyze" button  
- Wait a moment as GovOn reads and processes the request  
- The results will show on the screen  

### Review Results

- GovOn provides key points and suggestions  
- You can copy or save these results for your records

The app works offline after the first setup. You do not need internet to analyze new requests.
